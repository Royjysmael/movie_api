Roy's 90s Movie API

This RESTful API lets you explore iconic films from the 1990s. Use it to retrieve details about movies, directors, and genres via simple HTTP requests.

Built with Node.js, this project is designed for developers who want to build or experiment with 90s-themed movie data.

Endpoint Details

GET /movies

Returns a list of all movies in the database.

Method: GET

Response:

[
  {
    "title": "The Matrix",
    "year": 1999
  },
  ...
]

GET /movies/:title

Returns a single movie by title.

Example: /movies/Jumanji

Method: GET

Response:

{
  "title": "Jumanji",
  "year": 1995
}

GET /genres/:genreName

Returns a genre by name.

Example: /genres/Thriller

Method: GET

Response:

{
  "name": "Thriller",
  "description": "Suspenseful plots with twists and tension"
}

GET /directors/:directorName

Returns director data by name.

Example: /directors/Quentin Tarantino

Method: GET

Response:

{
  "name": "Quentin Tarantino",
  "bio": "Known for stylized storytelling and nonlinear plots.",
  "birthYear": 1963
}

POST /users

Registers a new user.

Method: POST

Request Body:

{
  "username": "roy",
  "email": "roy@royj.dev"
}

Response: New user registered

PUT /users/:username

Updates a user's info.

Example: /users/roy

Method: PUT

Request Body:

{
  "username": "roy_updated"
}

Response: Updated info for user: roy

POST /users/:username/movies/:movieId

Adds a movie to a user's favorites.

Method: POST

Example: /users/roy/movies/12345

Response: Added movie 12345 to roy's favorites

DELETE /users/:username/movies/:movieId

Removes a movie from a user's favorites.

Method: DELETE

Example: /users/roy/movies/12345

Response: Removed movie 12345 from roy's favorites

DELETE /users/:username

Deregisters an existing user.

Method: DELETE

Example: /users/roy

Response: Deleted user: roy
